Orientation — what this report measures
What this measures

A health score for one request host on a 0–100 scale. The host starts at 100 and loses points when mechanical signals — cache-miss rate, query-string churn, error rate, bot-share movement — cross thresholds. The current window is compared with the prior equivalent window.

How to read the score

Higher is healthier (100 = clean); lower = more triggered rules. Bands: escalate 0–40, monitor 40–70, observe 70–100.
